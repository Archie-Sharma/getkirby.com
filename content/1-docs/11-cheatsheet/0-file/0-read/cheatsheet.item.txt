Title:

{{class.instance}}->read($format = null)

----

Excerpt:

Reads the content of the file and returns it

----

Return:

type: string
text:

----

Text:

## Example

```php
// returns the content of the file
echo $page->file('readme.md')->read();

// parses the content as JSON
dump($page->file('data.json')->read('json'));
```
