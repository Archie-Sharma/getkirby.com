Title:

{{class.instance}}->hasPermission($target)

----

Excerpt:

Checks if the user is allowed to access the given target

----

Params:-
  name: $target
  type: string
  text:

----

Return:

type: boolean
text:

----

Text:

## Example

```php
<?php if($user = $site->user() and $user->hasPermission('panel')): ?>
This super secret content is only visible for users with panel access
<?php endif ?>
```
