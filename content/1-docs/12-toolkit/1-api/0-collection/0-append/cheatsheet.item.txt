Title: {{class.instance}}->append($key, $object)

Excerpt: Appends an element to the data array

text:

## Example

```
<?php
$collection = $pages;
$collection2 = $pages->children();
foreach($collection2 as $key => $item) {
  $collection->append($key, $item);
}
```
