Title: {{class.instance}}->filter($callback)

----

Excerpt: Filter the elements in the array by a callback function

----

Return:
type: $collection
text:

----

Params: -
  name: $callback
  type: function
  text: the callback function

----
Text:
```php
// fetch children with a title starting with 'Project'
$projects = $page->children()->filter(function($child) {
  return str::startsWith($child->title(), 'Project');
});

//fetch all images execept thoses that contain 'cover'
$images = $page->images()->filter(function($image) {
  return !str::contains($image->filename(), 'cover');
});
```
