Title: {{class.instance}}->filterBy($field [, $operator], $value [, $split])

----

Excerpt: Filters the current collection by a field, operator and search value

----
Params: -
  name: $field
  type: string
  text: Any field or method
-
  name: $operator
  type: string
  text: See list of filter methods below; if not a valid filter method the second parameter will be the value
-
  name: $value
  type: string
  text:
-
  name: $split
  type: string
  text: Character to split by while filtering

----

Return:
type: $collection
text:

----
Text:

## Available filter methods

Method   | Function
-        | -
==       | all values, which match exactly
!=       | all values, which don't match
*=       | all values, which contain the given string
>        | all values, which are greater than the given value
>=       | all values, which are greater or equal the given value
<        | all values, which are smaller than the given value
<=       | all values, which are smaller or equal the given value
