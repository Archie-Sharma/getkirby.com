Title: a::missing

----

Excerpt: Checks for missing elements in an array

----

Call: a::missing($array, $required = array())

----

Return: type: array
text: >
  An array of missing fields. If this is
  empty, nothing is missing.

----

Params: - 
  name: $array
  type: array
  text: The source array
- 
  name: $required
  type: array
  text: An array of required keys
