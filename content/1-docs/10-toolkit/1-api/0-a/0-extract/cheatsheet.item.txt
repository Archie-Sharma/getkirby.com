Title: a::extract

----

Excerpt: Extracts a single column from an array

----

Call: a::extract($array, $key)

----

Return: type: array
text: >
  The result array with all values from
  that column.

----

Params: - 
  name: $array
  type: array
  text: The source array
- 
  name: $key
  type: string
  text: The key name of the column to extract
